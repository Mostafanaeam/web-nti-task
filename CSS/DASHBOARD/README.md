# 📊 Admin Dashboard Project

A professional administrative interface designed for data management and user insights.

## 🌟 Key Features
- **Interactive Sidebar:** Seamless navigation between Dashboard, Clients, Products, Charts, and Settings.
- **User Profile:** Integrated profile section with user avatar and name.
- **Modern UI:** Clean, minimalist design for better focus on data.
- **Iconography:** Uses Font Awesome for intuitive navigation icons.

## 🛠️ Technologies
- **HTML5:** Modular page structure.
- **CSS3:** Custom styles and sidebar layout.
- **Font Awesome:** Professional icon set.

## 🚀 How to Run
Open `index.html` in any modern web browser to view the dashboard interface.
