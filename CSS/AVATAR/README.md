# 🎬 AVATAR Movie Project

A visually immersive movie landing page concept inspired by the "Avatar" cinematic universe.

## 🌟 Key Features
- **Immersive Header:** A clean navigation bar with a direct brand logo.
- **Cinematic Feel:** Focused on high-quality visuals and movie-themed layout.
- **Navigation:** Home, Films, About Us, and Contact Us sections.
- **Responsive Foundation:** Built with clean HTML and CSS for a professional look.

## 🛠️ Technologies
- **HTML5:** Semantic structure.
- **CSS3:** Custom styling and layout.

## 🚀 How to Run
Simply open `index.html` in your web browser.
