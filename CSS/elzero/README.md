# 📂 Elzero Web School Tasks Landing Page

A dedicated landing page inspired by the Elzero Web School, designed to organize and present various CSS assignments.

## 🌟 Key Features
- **Assignment Grid:** Organized cards showing lessons, assignment counts, and solution status.
- **Top Navigation:** Links to tracks, plans, and contact information.
- **Brand Integration:** Features the Elzero logo and thematic colors.
- **Clean Layout:** Focuses on educational content organization.

## 🛠️ Technologies
- **HTML5:** Structured for content clarity.
- **CSS3:** Flexbox/Grid for organized card layouts.

## 🚀 How to Run
Open `index.html` to browse through the educational assignments.
