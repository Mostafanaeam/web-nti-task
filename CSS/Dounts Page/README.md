# 🍩 Donuts E-commerce Storefront

An appetizing and vibrant landing page for a premium donut shop, featuring a "Bake It Yourself" vs "Buy Ready" concept.

## 🌟 Key Features
- **Dynamic Hero Section:** Dual call-to-action for DIY vs Ready-made donuts.
- **Product Showcase:** Color-coordinated product display with "Our Products" section.
- **Store Info:** Contact details and location information.
- **Newsletter Signup:** Integrated email subscription section.
- **Premium Typography:** Utilizes Google Fonts (Poppins) for a modern feel.

## 🛠️ Technologies
- **HTML5:** Semantic e-commerce structure.
- **CSS3:** Advanced layouts, transitions, and responsive design.
- **Google Fonts:** Poppins typeface.
- **Font Awesome:** Social media and UI icons.

## 🚀 How to Run
Open `index.html` to experience the sweet interactive design.
