# Hello There - Leon Agency Template

## 📝 About the Project
**Hello There** is a modern, minimal, and creative agency web template named **Leon**. This project was built as part of the Web Design training tasks at NTI Aswan to demonstrate proficiency in CSS layout techniques, responsive design, and component-based structure.

### 🚀 Key Features
- **Responsive Navigation:** A clean header with a mobile-friendly menu.
- **Landing Section:** High-impact "Hello There" landing area with creative typography.
- **Services Section:** Grid-based layout showcasing various agency services like Graphic Design, UI/UX, and Web Development.
- **Portfolio Gallery:** A structured showcase for creative projects.
- **About & Contact:** Sections dedicated to company info and communication channels.
- **Custom Styling:** Built using pure CSS with a focus on clean, maintainable code.

## 🛠️ Technologies Used
- **HTML5:** For semantic structural markup.
- **CSS3:** For custom layouts, styling, and animations (using a modular approach with `leon.css` and `normaliz.css`).
- **Font Awesome:** For professional iconography.
- **Google Fonts:** Specifically the "Work Sans" typeface for modern typography.

## 📂 Project Structure
- `index.html`: The main entry point for the template.
- `css/`: Contains custom styling and normalization files.
- `image/`: Stores all visual assets used in the project.
- `webfonts/`: Font Awesome local files.
