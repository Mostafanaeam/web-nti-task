# Kasper Template - Creative Portfolio

## 📝 About the Project
**Kasper** is a multi-page, professional creative portfolio template designed to showcase services, gallery items, and corporate information with a modern, artistic flair. This project was developed as a core task during the Web Design training at NTI Aswan, focusing on advanced CSS techniques, layout precision, and responsive components.

### 🚀 Key Features
- **Dynamic Landing Page:** Features an overlay with centered artistic messaging and background navigation.
- **Service Showcase:** A clean grid layout using Font Awesome icons to highlight key service offerings.
- **Featured Design Section:** A dedicated area for mobile-focused design showcases with custom list styling.
- **Interactive Portfolio:** A filtered project gallery with hover effects and detailed captions.
- **Video Integration:** A full-width background video section for high-impact visual storytelling.
- **Stats & Testimonials:** Data-driven statistics and a responsive customer testimonial slider.
- **Professional Skills Section:** Custom-built progress bars to visualize technical expertise.

## 🛠️ Technologies Used
- **HTML5:** For comprehensive, semantic document structure.
- **CSS3:** For advanced layouts (Flexbox, Grid), overlays, and custom animations.
- **Font Awesome:** For a rich, scalable icon system.
- **Google Fonts:** Utilizing "Open Sans" for its readability and professional aesthetic.

## 📂 Project Structure
- `index.html`: The complete structural markup for the landing page and all sections.
- `css/kasper.css`: Custom-coded styles including variables and responsive queries.
- `image/`: High-quality assets including project shots, logos, and section backgrounds.
